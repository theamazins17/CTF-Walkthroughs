# great-scott - easy

## Description
* Decode the coded message in the jpg file!

## Hint
* Use a steghide program

## Solution
* Download the file
* Using steghide in terminal, enter this command: `steghide extract -sf great-scott.jpg`
* Now type: `cat flag.txt`
* You've got the flag
* Flag: `nicc{It's_All_About_the_Mets!}`
